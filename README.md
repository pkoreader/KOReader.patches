# KOReader.patches

2-star-rating-hardcover.lua
It takes your hardcover linked book’s ratings and use it to show the star overlay on the respective books. I took idea from https://github.com/ImSoRight/KOReader.patches#2-star-rating-overlay. 
PS. For this to work, you need to have hardcover plugin installed and need to have your books linked.
